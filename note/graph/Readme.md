# Graph 筆記演進與優化

## 演進過程
1. **初版 (Representation)**: 學習如何用鄰接矩陣 (Adjacency Matrix) 和鄰接串列 (Adjacency List) 表示圖。
2. **演算法引入 (Traversal)**: 加入了 BFS 與 DFS 遍歷演算法。
3. **優化 (Shortest Path)**: 整合了 Dijkstra 與 Prim 演算法，並探討了如何利用優先權佇列 (Heap) 進行優化。

## 修改與優化
- **複雜度精確化**: 針對不同表示法下的演算法複雜度進行了詳細推導。
- **應用場景擴展**: 增加了拓撲排序與強連通分量的進階主題。
