# Queue & Stack 筆記演進與優化

## 演進過程
1. **初版 (LIFO & FIFO)**: 掌握堆疊與佇列的基本定義。
2. **實作多樣化**: 比較了使用 Array 與 Linked List 實作這兩種結構的優缺點。
3. **優化 (Advanced Applications)**: 加入了單調棧 (Monotonic Stack) 與單調隊列 (Monotonic Queue) 的進階技巧。

## 修改與優化
- **遞迴關係**: 強化了 Stack 與遞迴、系統呼叫棧 (System Call Stack) 之間的聯繫說明。
- **程式碼範例**: 增加了循環隊列 (Circular Queue) 的高效實作範例。
