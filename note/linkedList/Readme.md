# Linked List 筆記演進與優化

## 演進過程
1. **初版 (Singly Linked List)**: 專注於單向鏈結串列的指標操作。
2. **擴充 (Doubly & Circular)**: 增加了雙向鏈結串列與循環鏈結串列的內容，解決了單向串列無法反向遍歷的問題。
3. **優化 (Real-world Apps)**: 引入了 LRU Cache 等實際應用案例，展示鏈結串列在頻繁插入/刪除場景下的優勢。

## 修改與優化
- **指標邏輯強化**: 增加了更多圖解說明指標變動的過程，減少理解誤差。
- **效能對比**: 明確標註了與 Array 在不同場景下的效能權衡 (Trade-offs)。
