# Hash Table 筆記演進與優化

## 演進過程
1. **初版 (Hash Functions)**: 學習基本的雜湊函數設計。
2. **衝突處理 (Collision)**: 深入研究了 Chaining 與 Open Addressing (Linear Probing, Quadratic Probing) 等解決衝突的方法。
3. **優化 (Dynamic Resizing)**: 加入了 Load Factor 與動態擴容的機制說明。

## 修改與優化
- **安全性探討**: 增加了關於雜湊碰撞攻擊 (Hash Flooding) 的安全筆記。
- **現代實作參考**: 參考了 Java `HashMap` 的紅黑樹優化機制進行內容更新。
