# Array 筆記演進與優化

## 演進過程
1. **初版 (Basic Concepts)**: 最初僅記錄了陣列的定義與基本操作（存取、修改）。
2. **優化 (Performance Analysis)**: 加入了時間複雜度分析，強調 $O(1)$ 隨機存取的優勢。
3. **進階 (Memory Layout)**: 深入探討了陣列在記憶體中的連續性，以及這如何影響快取效率 (Cache Locality)。

## 修改與優化
- **結構化整理**: 將基本操作與進階應用（如動態陣列實作）分開。
- **視覺化表格**: 增加了與其他線性結構的對比表格。
- **程式碼重構**: 更新了範例程式碼，使其更符合現代 C++ 標準。
