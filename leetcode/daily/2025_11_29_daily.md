# LeetCode Daily 2025/11/29 筆記

## 題目：Make Sum Divisible by P (LeetCode 1590)

### 題目描述
- 給定整數陣列 nums 和整數 p
- 移除最短的子陣列，使剩餘元素和能被 p 整除
- 如果不可能，返回 -1

---

### 觀察
- 計算總和模 p 的餘數作為目標
- 使用前綴和和 hash map 記錄每個餘數的位置
- 找出最短的子陣列，其和的餘數等於目標

---

### 程式碼 (C++)

```cpp
using LL = long long;
class Solution {
public:
    int minSubarray(vector<int>& nums, int p) {
        LL sum = accumulate(nums.begin(), nums.end(), 0LL);
        if(sum % p == 0) return 0;
        int mini = nums.size();
        int target = sum % p;
        int nowRemain = 0;
        unordered_map<int, int> cnt;
        cnt[0] = -1;
        for(int i = 0; i < nums.size(); i++) {
            nowRemain = (nowRemain + nums[i]) % p;
            int demand = (nowRemain - target + p) % p;
            if( cnt.count(demand) ) {
                mini = min(mini, i - cnt[(nowRemain - target + p) % p]);
            }
            cnt[nowRemain] = i;
        }
        return (mini == nums.size()) ? -1 : mini;
    }
};
```
