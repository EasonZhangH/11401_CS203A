# LeetCode Daily 2025/11/17 筆記

## 題目：1-bit and 2-bit Characters (LeetCode 717)

### 題目描述
- 給定二進位陣列 bits
- 1 可單獨作為一個字元，10 和 11 是兩位元字元
- 判斷最後一個字元是否為單一位元字元

---

### 觀察
- 從頭開始解析：遇到 1 跳 2 步，遇到 0 跳 1 步
- 如果最後正好停在最後一位，說明最後是單一位元字元

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    bool isOneBitCharacter(vector<int>& bits) {
        int n = bits.size();
        int j = 0;
        while(j < n) {
            if(bits[j] == 1)
                j+=2;
            else
                j++;
        }
        if(j == n) return true;
        else return false;
    }
};
```
