# LeetCode Daily 2025/11/19 筆記

## 題目：Set Intersection Size At Least Two (LeetCode 757)

### 題目描述
- 給定區間陣列 intervals
- 找出最小集合，使得每個區間至少包含 2 個集合中的元素

---

### 觀察
- 按右端點排序，相同時左端點大的優先
- 貪心選擇：優先選擇區間右端的兩個點
- 檢查新區間是否已被覆蓋

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int intersectionSizeTwo(vector<vector<int>>& intervals) {
        sort(intervals.begin(), intervals.end(), [](auto& a, auto& b){
            if(a[1] == b[1]) return a[0] > b[0];
            else return a[1] < b[1];
        });
        int res = 2;
        int p1 = intervals[0][1], p2 = intervals[0][1] - 1;
        for(int i = 1; i < intervals.size(); i++) {
            int l = intervals[i][0], r = intervals[i][1];
            if(p2 >= l)
                continue;
            else if(p1 < l) {
                res+=2;
                p1 = r;
                p2 = r - 1; 
            }
            else {
                res++;
                p2 = p1;
                p1 = r;
            }
        }
        return res;
    }
};
```
