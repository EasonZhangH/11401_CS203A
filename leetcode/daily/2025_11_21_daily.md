# LeetCode Daily 2025/11/21 筆記

## 題目：Minimum Operations to Make Array Elements Equal to Zero (LeetCode 3190)

### 題目描述
- 給定整數陣列 nums
- 每次操作可以將任意元素 +1 或 -1
- 求使所有元素能被 3 整除的最少操作次數

---

### 觀察
- 對每個元素，計算其模 3 的餘數
- 餘數為 1 或 2 時需要一次操作
- 累加所有需要操作的次數

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int minimumOperations(vector<int>& nums) {
        int sum = 0;
        for(int n : nums)
            if(n % 3 != 0) sum += 1;
        return sum;
    }
};
```
