# LeetCode Daily 2025/11/18 筆記

## 題目：Keep Multiplying Found Values by Two (LeetCode 2154)

### 題目描述
- 給定陣列 nums 和初始值 original
- 如果 original 存在於陣列中，將其乘以 2
- 重複此過程，返回最終值

---

### 觀察
- 使用 hash table 記錄陣列中的數字
- 持續檢查並更新 original
- 當 original 超過 1000 或不存在時停止

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int findFinalValue(vector<int>& nums, int original) {
        int cnt[1001] = {0};
        for(int i = 0; i < nums.size(); i++) { cnt[nums[i]] = 1;}

        while( original <= 1000 && cnt[original]) {
            original *= 2;
        }
        return original;
    }
};
```
