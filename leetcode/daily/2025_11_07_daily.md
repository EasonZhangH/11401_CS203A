# LeetCode Daily 2025/11/07 筆記

## 題目：Minimum One Bit Operations to Make Integers Zero (LeetCode 1611)

### 題目描述
- 給定整數 n
- 可以執行兩種操作來將 n 變為 0
- 求最少操作次數

---

### 觀察
- 使用 Gray Code 的性質
- 將二進位轉換為 Gray Code
- Gray Code 的值即為答案

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int minimumOneBitOperations(int n) {
        int gray = 0;
        while(n) {
            gray ^= n;
            n >>= 1;
        }
        return gray;
    }
};
```
