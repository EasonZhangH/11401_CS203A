# LeetCode Daily 2025/11/22 筆記

## 題目：Greatest Sum Divisible by Three (LeetCode 1262)

### 題目描述
- 給定整數陣列 nums
- 找出最大的子集和，使其能被 3 整除

---

### 觀察
- 計算總和，檢查餘數
- 記錄模 3 餘 1 和餘 2 的最小值
- 根據總和餘數，移除相應的最小值

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int maxSumDivThree(vector<int>& nums) {
        int sum = 0;
        int m1 = INT_MAX, m1b = INT_MAX;
        int m2 = INT_MAX, m2b = INT_MAX;

        for (int x : nums) {
            sum += x;
            int r = x % 3;

            if (r == 1) {
                if (x < m1) { m1b = m1; m1 = x; }
                else if (x < m1b) m1b = x;
            }
            else if (r == 2) {
                if (x < m2) { m2b = m2; m2 = x; }
                else if (x < m2b) m2b = x;
            }
        }

        int r = sum % 3;

        if (r == 0) return sum;

        if (r == 1) {
            int remove1 = m1;
            int remove2 = (m2 != INT_MAX && m2b != INT_MAX) ? m2 + m2b : INT_MAX;
            return sum - min(remove1, remove2);
        }

        if (r == 2) {
            int remove1 = m2;
            int remove2 = (m1 != INT_MAX && m1b != INT_MAX) ? m1 + m1b : INT_MAX;
            return sum - min(remove1, remove2);
        }

        return 0;
    }
};
```
