# LeetCode Daily 2025/11/28 筆記

## 題目：Minimum Operations to Exceed Threshold Value (LeetCode 3512)

### 題目描述
- 給定整數陣列 nums 和閾值 k
- 每次操作可以從陣列中移除一個元素並減少總和
- 求使總和小於等於 k 的最少操作次數

---

### 觀察
- 計算陣列總和
- 如果 k 大於總和，返回總和（需要移除所有元素）
- 否則返回總和對 k 取模的結果

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int minOperations(vector<int>& nums, int k) {
        int sum = accumulate(nums.begin(), nums.end(), 0);
        return (k > sum) ? sum : sum % k;
    }
};
```
