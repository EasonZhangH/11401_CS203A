# LeetCode Daily 2025/10/29 筆記

## 題目：Minimum Number of Increments on Subarrays to Form a Target Array (LeetCode 1526)

### 題目描述
- 給定目標陣列 target，初始陣列全為 0
- 每次操作可以選擇一個子陣列，將其中所有元素 +1
- 求最少需要多少次操作才能得到 target

---

### 觀察
- 貪心策略：從左到右掃描
- 當前數字比前一個數字大時，需要額外的操作
- 只需累加「上升的差值」即可

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int minNumberOperations(vector<int>& target) {
        int res = 0;
        int pre = 0;

        for(int& num : target) {
            res += max(num - pre, 0);
            pre = num;
        }

        return res;
    }
};
```
