# LeetCode Daily 2025/11/12 筆記

## 題目：Maximum Score of Splicings in a Binary String (LeetCode 3228)

### 題目描述
- 給定二進位字串 s
- 每次遇到 "10" 模式時可以執行操作
- 求最大操作次數

---

### 觀察
- 統計 1 的數量
- 當遇到 "10" 模式時，累加當前的 1 的數量
- 連續的 0 只計算一次

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int maxOperations(string s) {
        int cnt = 0, ans = 0;
        int n = s.size();

        for(int i = 0; i < n; i++) {
            if(s[i] == '1')
                cnt++;
            else if(i > 0 && s[i - 1] == '1')
                ans += cnt;
        }
        return ans;
    }
};
```
