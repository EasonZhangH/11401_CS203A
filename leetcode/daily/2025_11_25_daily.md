#  LeetCode Daily 2025/11/25 筆記

## 題目：Smallest Integer Divisible by K (LeetCode 1015)

### 題目描述
- 找出最小的「由 1 組成的整數」能被 `k` 整除的長度。
- 如果不存在，回傳 `-1`。

---

### 觀察
- 答案是各種由 1 組成的數字：`1, 11, 111, 1111, ...`
- 檢查最短可以用 `k`整除的全1數字。
- 不需要真的建構大數字，因為絕對會stackoverflow，所以關鍵是追蹤 **餘數**。
- 若 `k` 含有因數 2 或 5 → 永遠不可能整除(尾巴不可能是1) → 回傳 `-1`。

---

### 程式碼 (C++)
```cpp
class Solution {
public:
    int smallestRepunitDivByK(int k) {
        if(k % 2 == 0 || k % 5 == 0) return -1;
        if(k == 1) return 1;
        int len = 1;
        int i = 1;
        while(i != 0) {
            i = (i * 10 + 1) % k;
            len++;
        }
        return len;
    }
};
```
