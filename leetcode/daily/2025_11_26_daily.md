# LeetCode Daily 2025/11/26 筆記

## 題目：Maximum Subarray Sum With Length Divisible by K (LeetCode 3381)

### 題目描述
- 給定整數陣列 nums 和整數 k
- 找出長度為 k 的倍數的子陣列，使其和最大

---

### 觀察
- 使用前綴和
- 對每個模 k 的餘數類別，獨立計算最大子陣列和
- 對於同餘類，維護最小前綴和，計算當前前綴和與最小前綴和的差

---

### 程式碼 (C++)

```cpp
using LL = long long;
class Solution {
public:
    long long maxSubarraySum(vector<int>& nums, int k) {
        int n = nums.size();
        LL ans = LLONG_MIN;
        vector<LL> prefix(n + 1, 0);
        for(int i = 0; i < n; i++) prefix[i + 1] = prefix[i] + nums[i];

        for(int i = 0; i < k; i++) {
            long long _min = prefix[i];
            for(int j = i + k; j <= n; j += k) {
                ans = max(prefix[j] - _min, ans);
                _min = min(_min, prefix[j]);
            }
        }
        return ans;
    }
};
```
