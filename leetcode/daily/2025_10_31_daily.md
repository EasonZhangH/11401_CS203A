# LeetCode Daily 2025/10/31 筆記

## 題目：Delete Nodes From Linked List Present in Array (LeetCode 3217)

### 題目描述
- 給定一個鏈結串列和一個數字陣列 nums
- 刪除鏈結串列中所有值出現在 nums 中的節點
- 返回修改後的鏈結串列頭節點

---

### 觀察
- 使用 hash table 記錄需要刪除的數字
- 使用 dummy node 簡化邊界處理
- 遍歷鏈結串列，遇到需要刪除的節點就跳過

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    ListNode* modifiedList(vector<int>& nums, ListNode* head) {
        int log[100001] = {0};
        ListNode* nxt = head;
        int value = 0;
        ListNode dummy(value, nxt);
        ListNode* prev = &dummy;
        
        for(int n : nums) {
            log[n] = 1;
        }
        
        while(prev->next != nullptr) {
            if(log[prev->next->val] == 1) {
                deleteNode(prev, prev->next);
            } else {
                prev = prev->next;
            }
        }
        
        return dummy.next;
    }

private:
    void deleteNode(ListNode* prev, ListNode* target) {
        prev->next = target->next;
        target->next = nullptr;
    }
};
```
