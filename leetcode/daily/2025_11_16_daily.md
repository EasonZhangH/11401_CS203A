# LeetCode Daily 2025/11/16 筆記

## 題目：Check If All 1's Are at Least Length K Places Away (LeetCode 1437)

### 題目描述
- 給定二進位陣列 nums 和整數 k
- 檢查所有 1 之間的距離是否至少為 k

---

### 觀察
- 記錄上一個 1 的位置
- 檢查相鄰兩個 1 之間的距離
- 如果距離小於 k+1，返回 false

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    bool kLengthApart(vector<int>& nums, int k) {
        int n = nums.size(), cur = 0;
        bool first = true;
        for(int i = 0; i < n; i++) {
            if(nums[i] == 1) {
                if(!first) {
                    if ( (i - cur) - k - 1 < 0) return false;
                }
                first = false;
                cur = i;
            }
        }
        return true;
    }
};
```
