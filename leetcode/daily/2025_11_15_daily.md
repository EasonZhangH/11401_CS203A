# LeetCode Daily 2025/11/15 筆記

## 題目：Number of Substrings With Only 1s (LeetCode 1513)

### 題目描述
- 給定二進位字串 s
- 計算只包含 1 的子字串數量
- 答案需取模 10^9 + 7

---

### 觀察
- 長度為 n 的連續 1，可組成 n*(n+1)/2 個子字串
- 遍歷字串，累加每段連續 1 的貢獻

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    long long subset(long long i) {
        return i * (i + 1) / 2;
    }
    int numSub(string s) {
        int n = s.size();
        const int MOD = 1e9 + 7;
        int len = 0;
        long long cnt = 0;
        for(int i = 0; i < n; i++) {
            if(s[i] =='1')
                len++;
            else {
                cnt += subset(len);
                cnt %= MOD;
                len = 0;
            }
        }
        cnt += subset(len);
        cnt %= MOD;
        return (int)cnt;
    }
};
```
