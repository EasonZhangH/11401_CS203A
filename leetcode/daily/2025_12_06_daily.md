# LeetCode Daily 2025/12/06 筆記

## 題目：Count Odd Numbers in an Interval Range (LeetCode 1523)

### 題目描述
- 給定區間 [low, high]
- 計算區間內奇數的數量

---

### 觀察
- 使用數學公式：1 到 n 的奇數數量為 (n+1)/2
- 答案為 [1, high] 的奇數數量減去 [1, low-1] 的奇數數量
- 可以用位運算優化

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int countOdds(int low, int high) {
        return ((high + 1) >> 1) - (low >> 1);
    }
};
```
