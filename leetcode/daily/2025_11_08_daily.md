# LeetCode Daily 2025/11/08 筆記

## 題目：Count Operations to Obtain Zero (LeetCode 2169)

### 題目描述
- 給定兩個整數 num1 和 num2
- 每次操作將較大的數減去較小的數
- 求需要多少次操作使其中一個數變為 0

---

### 觀察
- 遞迴實現
- 每次操作計數器加 1
- 直到其中一個數為 0

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int cnt = 0;
    int countOperations(int num1, int num2) {
        if(num1 == 0 || num2 == 0) return cnt;
        int b = max(num1, num2), s = min(num1, num2);
        cnt++;
        return countOperations(s, b - s);
    }
};
```
