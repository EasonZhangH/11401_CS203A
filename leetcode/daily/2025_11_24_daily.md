# LeetCode Daily 2025/11/24 筆記

## 題目：Paths in Matrix Whose Sum Is Divisible by K (LeetCode 2435)

### 題目描述
- 給定 m×n 矩陣和整數 k
- 從左上角到右下角，計算路徑和能被 k 整除的路徑數量
- 只能向右或向下移動

---

### 觀察
- 三維 DP：dp[r][c][mod] 表示到達 (r,c) 且路徑和模 k 為 mod 的方案數
- 轉移方程：從上方和左方轉移
- 答案為 dp[M-1][N-1][0]

---

### 程式碼 (C++)

```cpp
using LL = long long;
class Solution {
public:
    int numberOfPaths(vector<vector<int>>& grid, int k) {
        const int MOD = 1e9 + 7;
        int M =  grid.size(), N = grid[0].size();
        vector<vector<vector<int>>> dp(M, vector<vector<int>>(N, vector<int>(k, 0)));

        dp[0][0][grid[0][0] % k] = 1;
        for(int r = 0; r < M; r++)
            for(int c = 0; c < N; c++){
                if(r == 0 && c == 0) continue;
                for(int i = 0; i < k; i++) {
                    int cur = (grid[r][c] + i) % k;
                    LL fromU = (r >= 1) ? dp[r - 1][c][i] : 0;
                    LL fromL = (c >= 1) ? dp[r][c - 1][i] : 0;

                    dp[r][c][cur] = (fromU + fromL) % MOD;
                }
            }
        return dp[M - 1][N - 1][0];
    }
};
```
