# LeetCode Daily 2025/11/02 筆記

## 題目：Minimum Time to Remove All Balloons (LeetCode 1578)

### 題目描述
- 給定字串 c 和成本陣列 t
- 移除連續相同顏色的氣球，使得沒有連續相同的氣球
- 移除成本為 t[i]，求最小總成本

---

### 觀察
- 對於連續相同的氣球，保留成本最大的，移除其他的
- 貪心策略：遇到相同的氣球，累加較小的成本
- 更新最大成本，作為下次比較的基準

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int minCost(string c, vector<int>& t) {
        int res = 0, maxT = t[0];
        for (int i = 1; i < c.size(); ++i) {
            if (c[i] == c[i - 1]) {
                res += min(maxT, t[i]);
                maxT = max(maxT, t[i]);
            }
            else {
                maxT = t[i];
            }
        }
        return res;
    }
};
```
