# LeetCode Daily 2025/11/10 筆記

## 題目：Ones and Zeroes (LeetCode 474)

### 題目描述
- 給定字串陣列 strs 和兩個整數 m、n
- 找出最大子集大小，使得 0 的數量不超過 m，1 的數量不超過 n

---

### 觀察
- 01 背包問題
- 二維 DP：dp[i][j] 表示最多 i 個 0 和 j 個 1 能組成的最大子集
- 倒序遍歷避免重複使用

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int findMaxForm(vector<string>& strs, int m, int n) {
        vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));

        for (auto &s : strs) {
            int cur1 = count(s.begin(), s.end(), '1');
            int cur0 = s.size() - cur1;

            for (int i = m; i >= cur0; i--) {
                for (int j = n; j >= cur1; j--) {
                    dp[i][j] = max(dp[i][j], dp[i - cur0][j - cur1] + 1);
                }
            }
        }

        return dp[m][n];
    }
};
```
