# LeetCode Daily 2025/10/28 筆記

## 題目：Find the Integer Added to Array to Make Elements of Array Distinct (LeetCode 3370)

### 題目描述
- 給定一個整數 n，找出最小的整數，其二進位表示全為 1，且大於等於 n。
- 返回符合條件的最小整數。

---

### 觀察
- 答案是 2^len - 1 的形式（如 1, 3, 7, 15, 31...）
- 先計算 n 的位元長度
- 如果 n 已經是全 1 的形式，直接返回
- 否則返回下一個全 1 的數字

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int bit_length(unsigned int a) { 
        return a > 0 ? static_cast<int>(log2(a)) + 1 : 0;
    }
    int smallestNumber(int n) {
        int len = bit_length(n);
        if(n == (1<<len) - 1) return n;
        else return (1<<len) - 1;
    }
};
```
