# LeetCode Daily 2025/10/30 筆記

## 題目：The Two Sneaky Numbers of Digitville (LeetCode 3289)

### 題目描述
- 給定陣列 nums，其中有兩個數字出現兩次，其餘數字各出現一次
- 找出這兩個重複的數字

---

### 觀察
- 使用 XOR 位元運算
- 先對所有數字和期望的數字做 XOR，得到兩個重複數字的 XOR 結果
- 利用最低位的 1 來分組，分別找出兩個數字

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    vector<int> getSneakyNumbers(vector<int>& nums) {
        int n = nums.size();
        int total = 0;

        for(int x : nums) {total ^= x;}
        for(int i = 0; i < n-2; i++) {total ^= i;}

        int lowestBit = (total) & (-total);
        int a = 0, b = 0;

        for(int x : nums) {
            if(x & lowestBit) { a ^= x; }
            else { b ^= x; }
        }

        for (int i = 0; i < n - 2; i++) {
            if (i & lowestBit) a ^= i;
            else b ^= i;
        }
        return {a, b};
    }
};
```
