# LeetCode Daily 2025/12/03 筆記

## 題目：Count Collisions on a Road (LeetCode 2211)

### 題目描述
- 給定字串 directions 表示車輛方向（L/R/S）
- 計算總共發生的碰撞次數
- L 表示向左，R 表示向右，S 表示靜止

---

### 觀察
- 移除開頭的 L（直接離開道路）
- 移除結尾的 R（直接離開道路）
- 中間所有非 S 的車都會碰撞

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int countCollisions(string directions) {
        int n = directions.size(), collisions = 0;
        int i = 0, j = n - 1;
        while(i < n && directions[i] == 'L')
            i++;
        while(j >= 0 && directions[j] == 'R')
            j--;
        if(j <= i) return 0;
        for(int idx = i; idx < j + 1; idx++)
            if(directions[idx] != 'S')
                collisions++;
        return collisions;
    }
};
```
