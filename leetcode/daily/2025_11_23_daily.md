# LeetCode Daily 2025/11/23 筆記

## 題目：Binary Prefix Divisible By 5 (LeetCode 1018)

### 題目描述
- 給定二進位陣列 nums
- 對每個前綴，檢查其代表的二進位數是否能被 5 整除

---

### 觀察
- 維護當前數字模 5 的餘數
- 每次左移一位並加上新位元
- 更新餘數並檢查是否為 0

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    vector<bool> prefixesDivBy5(vector<int>& nums) {
        int cur_mod = 0;
        vector<bool> ans;
        for(int n : nums) {
            cur_mod = ( (cur_mod << 1) + n) % 5;
            ans.push_back(cur_mod == 0);
        } 
        return ans;
    }
};
```
