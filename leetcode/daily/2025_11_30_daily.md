# LeetCode Daily 2025/11/30 筆記

## 題目：Maximum Running Time of N Computers (LeetCode 2141)

### 題目描述
- 給定 n 台電腦和電池陣列
- 電池可以在電腦間切換
- 求所有電腦同時運行的最長時間

---

### 觀察
- 二分搜尋答案（運行時間）
- 對每個時間，檢查是否有足夠的電量支撐
- 每個電池貢獻 min(time, battery) 的電量

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    long long MOD = 1e14;
    bool isValid(long long time, int n, vector<int>& batteries) {
        long long total = 0;
        for(int b :batteries) {
            total += min(time, (long long)b);
        }
        return total >= (time * n) % MOD;
    }
    long long maxRunTime(int n, vector<int>& batteries) {
        long long low = 0;
        long long high = accumulate(batteries.begin(), batteries.end(), 0LL) / n;

        while(low < high) {
            long long mid = (low + high + 1) / 2;
            if(isValid(mid, n, batteries)) {
                low = mid;
            }
            else
                high = mid - 1;
        }
        return low;
    }
};
```
