# LeetCode Daily 2025/11/09 筆記

## 題目：Minimum Operations to Make Array Non-decreasing (LeetCode 3542)

### 題目描述
- 給定陣列 nums
- 可以刪除任意元素
- 求最少操作次數使陣列非遞減

---

### 觀察
- 使用單調堆疊維護遞增序列
- 遇到小於堆疊頂端的元素時彈出
- 加入新元素時操作數加 1

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    int minOperations(vector<int>& nums) {
        stack<int> s;
        int ops = 0;

        for(int num : nums) {
            while(!s.empty() && num < s.top() )
                s.pop();
            if(num > 0 &&(s.empty() || num > s.top())) {
                s.push(num);
                ops++;
            }
        }
        return ops;
    }
};
```
