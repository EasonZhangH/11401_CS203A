# LeetCode Daily 2025/11/13 筆記

## 題目：Increment Submatrices by One (LeetCode 2536)

### 題目描述
- 給定 n×n 矩陣，初始全為 0
- 每個查詢指定一個子矩陣範圍，將其中所有元素 +1
- 返回最終矩陣

---

### 觀察
- 使用差分陣列優化
- 對每一行使用差分標記
- 最後進行前綴和還原

---

### 程式碼 (C++)

```cpp
class Solution {
public:
    vector<vector<int>> rangeAddQueries(int n, vector<vector<int>>& queries) {
        vector<vector<int>> ans(n, vector<int>(n, 0));
        for(vector<int>& q : queries)
            for(int r = q[0]; r <= q[2]; r++) {
                ans[r][q[1]]++;

                if(q[3] + 1 < n)
                    ans[r][q[3] + 1]--; 
            }
        
        for(int r = 0; r < n; r++)
            for(int c = 1; c < n; c++)
                ans[r][c] += ans[r][c - 1];

        return ans;
    }
};
```
